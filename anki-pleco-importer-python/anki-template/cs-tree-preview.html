How it works:
  • 🏗️ Tree Structure: Each level is a `tree-level` div containing nodes
  • 📦 Tree Nodes: Individual `tree-node` divs positioned spatially
  • 🌿 Tree Branches: Group sibling nodes with `tree-branch` divs
  • 🔗 Connecting Lines: CSS automatically draws lines between levels
  • 🎨 Level Colors: Each level gets different colors (level-1, level-2, etc.)
  • 📱 Responsive: Adapts to mobile with smaller nodes
