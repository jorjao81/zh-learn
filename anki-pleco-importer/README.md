# @jorjao81/anki-pleco-importer

- Improves pleco flashcards export for better formatting in Anki

## Required

- Node.js 14+

## How to using

- Lunch commands
  - `npm start`
    - run cli
  - `npm run dev`
    - run force debug cli
  - `npm run type-check`
    - run type-check
  - `npm run build`
    - run swc
      - this only development, not production
  - `npm run test`
    - launches test runner by watch mode
  - `npm run coverage`
    - ~~get coverage report~~
